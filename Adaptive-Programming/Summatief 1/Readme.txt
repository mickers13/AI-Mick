In deze map heb ik een voorbeeld van een hout plantage getekend. 

Hierin kun je zien dat het nogal gefocusseerd is rond een bedrijf genaamd de Banjo.

Hier worden 3 soorten hout verbouwd, waar er maximaal 2 per bos passen. 

Verder heb je een hout hakker die bomen om kan hakken.

Ook heb je boomplanters, die de bomen kunnen planten, deze houden dus ook regel met de bovenstaande businessrule.


Verder zijn specifieke dingen nog verder toegelicht in de UML diagram als notitie.

De medewerkers maken gebruik van een interface, en de bomen doen dat ook van een andere. 

Ik moet zeggen dat ik dit een vrij uitdagende opdracht vond, omdat mijn creativiteit redelijk op de proef werd gesteld. 
Maar helaas kwam ik er op de laatste dag achter dat mijn originel UML design Véél te ambitious was om goed werkend, en netjes te krijgen. 
Hierdoor heb ik veel (halfwerkende) code en andere classen moeten schrappen aangezien ik liever een werkend programma op zou willen leveren, en ik nogsteeds
voldoe aan alle opdracht's eisen. Ik heb de originele versie van het UML diagram ook ingeleverd, als "TEUITGEBREID NIET BEOORDELEN". Dit
 zodat jullie kunnen kijken wat ik origineel vanplan was.
